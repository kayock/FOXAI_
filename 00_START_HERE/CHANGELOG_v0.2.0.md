# KayocktheOS v0.2.0 - Bridge Desktop

## Changed
- Promotes the browser home screen into the KayocktheOS Bridge Desktop.
- Adds a Bridge button to the Hunter sidebar.
- Adds live Core API status cards to the browser home screen.
- Adds `kayock://bridge` as an alias for the Bridge home.
- Adds `kayock://workshop` routing.
- Updates CSP so the browser can read the local Core API at `http://127.0.0.1:8844`.
