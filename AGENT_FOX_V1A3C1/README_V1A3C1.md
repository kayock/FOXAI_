# Agent Fox Technical Core V1A-3C1

This isolated component builds one bounded static dependency closure for the protected Web Portable path:

- launcher: `Z:\FOXAI\START_FOXAI_WEB_PORTABLE.bat`
- interpreter evidence: `Z:\FOXAI\env\python\python.exe`
- entry script: `Z:\FOXAI\core\foxai_web.py`

The builder verifies prior evidence receipts, selects exactly one import context, computes a preliminary closure from verified indexes, then reads only reached first-party source files. Each reached source is read at most once, hash-checked, parsed sequentially, and released. No raw source content is written into the evidence.

The payload does not execute FOXAI source, launchers, imports, packages, models, ComfyUI, Llama, PowerShell, or child processes. Static provider evidence never proves an import succeeded or ran.
