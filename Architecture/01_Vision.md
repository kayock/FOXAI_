# FOXAI Project Orion Vision

FOXAI Command OS is a portable, offline-first AI operating environment.

It is not one chatbot.

It is an organized system of:
- Kernel
- Fleet
- Departments
- Officers
- Professors
- Shuttles
- Tools
- Local knowledge
- Mission history

Guiding principle:

> Every subsystem has a purpose. Every purpose has a steward. Every steward reports to the Bridge.

FOXAI exists to edify: to help users learn, create, repair, understand, and build.
