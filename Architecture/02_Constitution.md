# FOXAI Constitution

## Core Rule

Nothing is hard-coded if it can be discovered.
Nothing is duplicated if it can be described.
Nothing is hidden if it can report its own status.

## Protected Core

The following systems are protected architecture:

- FOXKernel
- MissionBus
- Fleet Registry
- Department Registry
- Extension Manager
- Academy API
- Bridge API
- Vault
- Captain's Log
- Repair Bay
- Shipyard

## Architectural Layers

1. User Interface
2. Bridge API
3. FOXKernel
4. MissionBus
5. Departments
6. Services
7. Tools
8. Data / Vault / Knowledge

## Separation Rule

Personality must remain separate from responsibility.

Example:

Chief Engineer Ada is the user-facing steward.
The Engineering Department is the service boundary.
Ruff, Black, MyPy, PyDeps, and Repair Bay are tools.
