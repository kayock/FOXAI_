# Bridge UI Specification

The Bridge is the user-facing command center.

It should display live system state from FOXKernel, not hard-coded values.

## Core Panels

- Fleet Status
- Mission Console
- Mission Queue
- Bridge Officers
- Captain's Log
- Engineering
- Science
- Security
- Communications
- Academy
- Creative Studio
- Repair Bay
- Iron Library

## UI Rule

The Bridge should ask FOXKernel what exists.

It should not assume departments exist.

## Accessibility

The Bridge should support:

- Keyboard navigation
- Large text
- High contrast
- Screen reader-friendly labels
- Reduced motion mode
