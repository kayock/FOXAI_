# Feature 002 - Bridge Application

The Bridge is the front door of KayocktheOS.

## Run

1. Start Core:

```bat
Start_KayocktheOS.bat
```

2. Start Bridge:

```bat
Start_Bridge.bat
```

## Rooms

- Home
- Academy
- Workshop
- Library
- Observatory
- Foundry

## Design rule

The Bridge is not a launcher. It is the operating environment.
