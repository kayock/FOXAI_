# KayocktheOS Launcher Notes

Recommended user-facing launcher:

```text
Z:\KayocktheOS\Launch_KayocktheOS_Workshop.bat
```

This launcher:

1. Checks whether the Core API is already online.
2. Starts the Core if needed.
3. Checks Bridge dependencies.
4. Opens the Bridge.

The goal is to make KayocktheOS feel like one application instead of a set of scripts.
