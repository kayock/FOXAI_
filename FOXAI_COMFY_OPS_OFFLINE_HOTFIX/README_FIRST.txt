FOXAI COMFYUI OFFLINE DISPLAY HOTFIX

This fixes the flashing raw URL timeout shown when ComfyUI is stopped.

1. Close FOXAI Desktop and WebUI.
2. Extract this entire folder directly into Z:\FOXAI.
3. Open Z:\FOXAI\FOXAI_COMFY_OPS_OFFLINE_HOTFIX.
4. Run APPLY_COMFY_OPS_OFFLINE_HOTFIX.bat.
5. Press Y once.
6. Start WebUI or Desktop normally.

Expected offline display:
COMFYUI OPERATIONS // OFFLINE
ComfyUI is stopped. Start it when you need Red Canvas.

The hotfix also reduces offline polling to once every 3 seconds.
It does not change ComfyUI, models, launchers, or generation.
