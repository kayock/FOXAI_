INSTALL

1. Close the FOXAI WebUI and its command window.
2. Extract this complete folder directly inside Z:\FOXAI.
3. Run:

   Z:\FOXAI\FOXAI_KAYOCKS_STUDY_INTEGRATION_V1_LIVE_COMPATIBLE\
   APPLY_KAYOCKS_STUDY_INTEGRATION_V1.bat

4. Press Y.
5. Restart the FOXAI WebUI once.

WHAT CHANGES

- Iron Library becomes Kayock's Study in the sidebar.
- The existing Library page becomes the Study home.
- Entering that page starts Bibliotheca only when needed.
- FOXAI shows live service, model, document, page, shelf, and duplicate totals.
- The full Bibliotheca can run embedded inside FOXAI.
- Open Full Study preserves the standalone room.
- The old filename browser and text index remain in a collapsed legacy section.

WHAT DOES NOT CHANGE

- Navigation grouping
- Kayock Writer
- Poetry Studio
- Repair Bay
- Engineering pages
- Bibliotheca server or database
- Original PDFs
- Standalone launcher
