FOXAI KAYOCK WRITER CALM HOME

PURPOSE

Replace the report-heavy Kayock Writer landing page with a calm creative home.

MAIN WRITER ROOMS

- Poetry Studio
- Story Forge
- World Builder
- My Writing

POETRY STUDIO

The first Poetry Studio build is functional:
- Theme
- Emotion
- Scene or imagery
- Voice
- Free verse, rhymed, sonnet, haiku, or spoken word
- Short, medium, or long
- Optional opening line
- Create Poem
- Try Another
- Editable result
- Save Draft
- Send to Poem Polisher
- Poem Polisher with selectable focus

Poems are generated by the local chat model already started in Artificial Minds.
Saving creates a new Markdown file under:

Projects\KayockWriter\Poetry\Drafts

Existing files are never overwritten.

WORLD BUILDER

Provides calm D&D-focused starting points:
- Create a World
- Build a Region
- Make a Settlement
- Create a Faction
- Design a Deity or Religion
- Build a Dungeon or Adventure Site
- Create a Campaign
- Open the existing Codex

World Builder currently prepares a guided Mission Console brief and asks FOXAI
to proceed one useful question at a time. It reuses the existing Codex rather
than creating a disconnected database.

SIDEBAR

The normal Kayock Writer group contains only five understandable destinations.
The many existing gates, reports, audits, and verification screens remain under
a collapsed Writer Advanced Tools group.

INSTALL

1. Close FOXAI WebUI.
2. Extract this entire folder directly into Z:\FOXAI.
3. Run APPLY_KAYOCK_WRITER_CALM_HOME.bat.
4. Press Y once.
5. Restart WebUI and open Kayock Writer.

Only core\foxai_web.py is replaced. A timestamped backup is created first.
