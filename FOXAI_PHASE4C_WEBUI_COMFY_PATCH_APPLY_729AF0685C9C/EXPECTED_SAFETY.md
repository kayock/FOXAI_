# Phase 4C-A Safety Contract

Authorized:

- Create one verified backup of the current `core\foxai_web.py`
- Replace only `core\foxai_web.py` with the exact approved source
- Verify hashes, size, Python syntax, protected files, and shortcuts
- Roll back only the approved target from the verified backup on failure

Forbidden:

- Modify any other live source or configuration file
- Change launchers or shortcuts
- Change ComfyUI source
- Delete preexisting user work
- Install or download packages
- Use network access
- Launch FOXAI, WebUI, ComfyUI, browser, models, or services
