SPECIAL DETAILS

Dust and Duty includes:
- old-west frontier atmosphere
- fantasy ruins and failing science-fiction machinery
- strange stars and impossible roads
- campfire stew in a blackened iron pot
- tin cups, quiet fellowship, and found family
- grief carried through actions rather than speeches

The Forsaken Flame uses the public-domain Frankenstein tradition while creating
new exiles, conflicts, imagery, and endings.

Beneath the Beats stores no copyrighted lyrics and does not imitate a specific
living or recently deceased performer. It uses broad, original spoken-word and
West Coast craft characteristics.

Poe uses public-domain literary craft traits but still creates new poems rather
than reproducing famous lines, titles, refrains, or plots.

INSTALL

1. Close FOXAI WebUI.
2. Extract this folder directly inside Z:\FOXAI.
3. Run APPLY_POETRY_HOLIDAY_VOICE_PACK_V1.bat.
4. Press Y.
5. Restart FOXAI WebUI.
6. Open Poetry Studio and choose a profile from Voice.

SAFETY

- Only core\foxai_web.py changes.
- The existing file is backed up first.
- The installer refuses to overwrite an unexpected version.
- Python syntax and pack markers are verified after installation.
- A rollback launcher is included.
- Existing poems and source poems are not modified.
- No generated alternative changes a poem until Use This Alternative is pressed.
