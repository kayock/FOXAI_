FOXAI RHYME COACH CLARITY FIX

WHAT WAS CONFUSING

Changing the dropdown from AABB to ABAB or AAAA did not change the displayed
analysis. That is expected because the poem text did not change, but the old
"Analyze Again" wording made it appear broken.

WHAT CHANGED

- "Analyze Again" is renamed "Recheck Poem."
- "Workshop rhyme target" is renamed "Revision target for the Workshop."
- A visible notice explains that analysis and revision are separate.
- Changing the target immediately explains how it will guide the next stanza
  revision.
- Rechecking an unchanged poem explicitly reports that the analysis is
  unchanged.
- Pressing Revise This Stanza still sends the selected AABB, ABAB, AAAA,
  natural-rhyme, or rhythm-only instruction to the Selection Workshop.

The analysis itself remains an honest description of the current poem. It does
not pretend the poem already follows a target that has not yet been applied.

INSTALL

1. Close FOXAI WebUI.
2. Extract this folder directly inside Z:\FOXAI.
3. Run APPLY_RHYME_COACH_CLARITY_FIX.bat.
4. Press Y.
5. Restart WebUI.

Only core\foxai_web.py changes after an automatic backup.
