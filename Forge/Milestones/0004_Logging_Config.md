# Milestone 0004 – Logging & Configuration

## Goal

Make KayocktheOS safer and easier to understand by giving the Core Launcher structured configuration, module ownership, and persistent boot logs.

## Success Criteria

- Core reads manifest.
- Core reads operator profile.
- Core reads module registry.
- Core creates logs automatically.
- Bridge can show module status.
- Bridge can show health report.
- Missing items are reported rather than hidden.
