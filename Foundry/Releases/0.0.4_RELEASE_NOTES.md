# KayocktheOS 0.0.4 – Logging & Configuration

This release improves the Core Launcher foundation.

## Added

- `System/Config/modules.yaml`
- `System/Config/health_policy.yaml`
- Better `launch.py`
- Boot logs
- Bridge menu health report
- Module registry view

## How to use

1. Unzip onto the USB drive.
2. Double-click `Start_KayocktheOS.bat`.
3. Choose a Bridge menu option.
