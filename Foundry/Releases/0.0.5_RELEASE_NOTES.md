# KayocktheOS 0.0.5 - Module Manager

This release introduces the Module Manager.

## Highlights

- Departments are now proper module records.
- Bridge reads module metadata from `System/Registry/modules/`.
- Module statuses are visible at startup.
- Enabled modules are checked for valid entry paths.

## Operator Notes

Unzip this release onto the USB as the next clean checkpoint. No model files are included.
