# KayocktheOS 0.0.6 Release Notes

## Codename
First Boot Wizard

## Purpose
Create the first personalization layer without hardcoding identity into the launcher.

## Added
- First Boot Wizard in `System/Launchers/launch.py`
- Operator Profile creation and update logic
- Runtime nickname greeting from `System/Config/operator.yaml`
- Bridge menu option to view the Operator Profile
- Bridge menu option to reset the First Boot Wizard
- Manifest paths now declare the Operator Profile location

## Engineering Decision
Documentation uses **Operator**. Runtime may use the chosen nickname.
