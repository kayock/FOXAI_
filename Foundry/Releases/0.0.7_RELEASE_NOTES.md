# KayocktheOS 0.0.7 Release Notes

## Codename
Browser Detection & Launch

## Summary
This release integrates Kayock Browser into the Core Launcher as a detectable and launchable module.

## Included
- Dynamic browser executable detection
- Bridge health check for browser availability
- Browser launch menu option
- Launch logging
- Updated module manifest and docs

## Not Included
The Kayock Browser executable is not bundled. Add your existing portable EXE to `Interface/Kayock_Browser`.
