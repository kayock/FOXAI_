# Phase 3D-A Safety Contract

Authorized operation: add only the exact approved 3,520 files.

Prohibited:

- Overwriting or modifying any existing file
- Deleting any preexisting file or directory
- Changing protected shortcuts
- Changing existing launchers or FOXAI source
- Changing Runtime/Core, Config, ComfyUI, Models, or protected baselines
- Installing or downloading packages
- Network access
- Launching FOXAI, ComfyUI, or a browser
- Running the new diagnostic during apply

Rollback is limited to add-only files created by the current run, and only while their hashes still match the approved plan.
