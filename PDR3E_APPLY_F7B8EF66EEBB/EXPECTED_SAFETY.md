# Phase 3E-A Safety Contract

Authorized: add one exact new launcher file.

Prohibited:

- Overwriting or modifying any existing file
- Deleting any preexisting file
- Changing either USB-root shortcut
- Changing existing launchers or FOXAI source
- Installing or downloading packages
- Network access
- Launching FOXAI, ComfyUI, or a browser
- Running the new launcher during apply

On failure after addition, rollback may remove only the newly created launcher while its hash still matches the approved file.
