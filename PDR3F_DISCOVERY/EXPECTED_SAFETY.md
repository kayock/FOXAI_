# Phase 3F Safety Contract

Allowed:

- Read exact known FOXAI launchers, shortcuts, and targeted source files
- Resolve shortcut metadata through read-only Windows PowerShell COM access
- Follow a bounded local launcher-script reference chain
- Write timestamped evidence only inside this package

Forbidden:

- Modify any live FOXAI file or shortcut
- Launch FOXAI, ComfyUI, browser, or model services
- Install or download packages
- Use network access
- Recursively scan drives
- Propose or apply a replacement launcher in this phase
