# KayocktheOS v0.1.2 - Dashboard Live View

This release improves the browser Bridge dashboard.

## What changed

- The Shell opens to the dashboard instead of raw JSON.
- Dashboard reads live status from the Core API.
- Health, modules, Shell status, Operator greeting, and timestamps update automatically.
- Added clearer Core/Shell status cards.
- Kept the API read-only and localhost-only.

## Quick Start

1. Keep or copy your Kayock Browser portable EXE into:

```text
Interface/Kayock_Browser/
```

or:

```text
Shell/Kayock_Browser/
```

2. Run:

```bat
Start_KayocktheOS.bat
```

3. Choose:

```text
4. Launch Kayock Browser Shell
```

The browser should open the Bridge dashboard.
