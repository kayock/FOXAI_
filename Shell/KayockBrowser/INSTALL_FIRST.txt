KAYOCK BROWSER 2.5.1 RC - FOUNDATION RELEASE

NOVICE INSTALL INSTRUCTIONS

1. Right-click the ZIP file and choose Extract All.
2. Open the extracted folder.
3. Double-click RUN_KAYOCK.bat.
4. If it says Node.js is missing, install Node.js LTS from https://nodejs.org.
5. After Node installs, double-click RUN_KAYOCK.bat again.

First launch may take a few minutes while dependencies install.

Do not use this alpha build for banking, taxes, or high-security accounts yet.
