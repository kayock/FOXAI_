# Known Issues — 2.5.1 RC

- Windows installer must be built/tested on Windows for the final `.exe` artifacts.
- App is not code-signed yet; Windows SmartScreen may warn on first launch.
- Some login providers may need popup handling tests after the security hardening pass.
- DevTools are intentionally disabled unless `KAYOCK_DEVTOOLS=1` is set.
- AI tools are intentionally not included in this browser foundation release.
