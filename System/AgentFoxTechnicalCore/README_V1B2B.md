Agent Fox Technical Core V1B-2B

Purpose
-------
Perform a read-only static and callable preflight for connecting the verified
V1B-2A resource-evidence provider to the existing shared self-knowledge route.

The preflight verifies the exact provider, evidence, bridge, adapter, helper,
WebUI, and Desktop hashes. It maps both live call chains, tests the proposed
routing order on both surfaces, checks all 18 resource categories for overlap,
and preserves protected-context priority and ordinary chat.

No integration is applied in this mission. No live system scan, process or
listener inspection, network or model call, GUI launch, Windows change, source
evidence change, or rollback-drive access is performed.
