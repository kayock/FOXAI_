# Forge Journal 0001 — Core Launcher Begins

KayocktheOS moved from planning into implementation with v0.0.2.

Major decisions:

- The project uses Operator in documentation.
- Runtime greetings use a configurable chosen name.
- The Bridge is the startup status experience.
- Core Launcher coordinates modules but does not contain module logic.
- Configuration lives in files, not hardcoded values.

This keeps the system personal without becoming locked to one person or one machine.
