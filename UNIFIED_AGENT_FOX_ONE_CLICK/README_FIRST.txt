FOXAI UNIFIED AGENT FOX CHAT — ONE CLICK

1. Close FOXAI if it is running.
2. Extract this folder directly into Z:\FOXAI.
3. Open the UNIFIED_AGENT_FOX_ONE_CLICK folder.
4. Double-click INSTALL_UNIFIED_AGENT_FOX_CHAT.bat.
5. Start FOXAI normally.

No Engineering Workshop command is used.
No PowerShell command is used.
The installer changes only Z:\FOXAI\core\director.py.

It makes a small backup before replacement and automatically restores the
original file if compilation or the live routing check fails.

- Ordinary sentences remain in Agent Fox chat.
